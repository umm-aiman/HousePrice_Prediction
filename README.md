# House Price Prediction

## Objective
Predict house prices using regression models and property features.

## Dataset
House Price Prediction Dataset

## Features Used
- Area
- Bedrooms
- Bathrooms
- Floors
- YearBuilt
- Location
- Condition
- Garage

## Models Used
- Linear Regression
- Gradient Boosting Regressor

## Evaluation Metrics
- MAE
- RMSE
- R2 Score

## Technologies
- Python
- Scikit-learn
- Pandas
- Matplotlib
- Seaborn

## Run Project

```bash
pip install -r requirements.txt
jupyter notebook
```
